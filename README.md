# Switcher

AWS switch-role page URL generator.

## install

1. download from [release page](https://github.com/reireias/switcher/releases)

2. unarchive

3. `sudo mv ./switcher /usr/local/bin`

## install from source

1. clone this repository

2. `make && make install`

## color
valid colors are follows

- red
- prange
- yellow
- green
- blue
